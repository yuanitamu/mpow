merupakan game berbasis java dengan platform Nokia SDK 2.0 for Java ME. sebelumnya game ini 
dikenal 
dengan nama whack the mole, dimana kami mengembangkan game ini dengan penambahan aturan 
dari kami dan icon-icon
yang berbeda dibuat khusus untuk anak TK usia 4-8 tahun. cara bermainnya cukup mudah, yaitu 
dengan mengklik icon jamur
yang akan muncul dari lubang. selain jamur, ada pula bom jebakan yang apabila diklik maka nyawa 
pemain akan berkurang.
reward dari game ini berupa piala dimana jumlah piala yang didapatkan tergantung pada berapa 
banyak poin yang
berhasil didapatkan.
